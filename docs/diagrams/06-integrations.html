คำอธิบาย

ทำไมต้อง mock-by-default — ทุกการเชื่อมต่อภายนอกเริ่มต้นเป็น mock ทำให้รัน/ทดสอบ/พัฒนาได้โดยไม่ต้องมี credential จริงหรือเสียค่า API แล้วค่อยพลิกเป็น real ทีละตัวด้วย env เมื่อพร้อม (ไม่ต้อง redeploy code)

ฝั่งเข้า vs ฝั่งออก — PeopleSoft เป็นสองทิศ: รับใบสมัคร/ตำแหน่งเข้ามา (Direction A, ตรวจ HMAC) และ sync พนักงานใหม่ออก (Direction B, OAuth Integration Broker)

สถานะ provider บน production ปัจจุบัน

  • real OpenAI, Doc Intelligence, LINE, Entra SSO, ACS Email
  • mock Microsoft Graph (Teams meeting จริงรอ Azure AD app + admin consent) · AI Search (phase-2)

โค้ด HR-as-attendee สำหรับนัดสัมภาษณ์พร้อมแล้ว แต่จะ active เมื่อเปิด GRAPH_PROVIDER=real
